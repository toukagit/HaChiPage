:::tip
程序在启动时称为**冷启动**，程序启动后切换到后台再切换回前台时称为**热启动**。
:::

## 展示开屏广告
由SDK控制开屏广告的加载和展示。

## 最佳做法
在冷启动时使用开屏广告的首选方法是，使用加载屏幕来加载游戏或应用素材资源，并且仅在加载屏幕展示广告。<br/>例如游戏有load场景，在收到SDK初始化回调时，关闭load场景，加载游戏场景。<br/>SDK初始化回调参见 [【SDK集成 - 初始化SDK】](integration.md)
