本页向您展示如何下载、导入HaChi Unity Plugin。

:::tip

 - **接入HaChiSDK 需要 Xcode 14.2 或更高版本**       
 - **更新新版本SDK最好先将老版本SDK删除，配置文件除外。并且导出新的Xcode或者Android Studio工程，注意不要覆盖导出到老版本SDK导出的工程。**   

:::


## 下载最新的Unity插件 
[HCSDK_2.3.5_57a32d5.unitypackage](https://touka-artifacts.oss-cn-beijing.aliyuncs.com/TKG%20%E5%8F%91%E8%A1%8C%E6%8A%80%E6%9C%AF/Hachi%20SDK/Unity/2.3.5/HCSDK_2.3.5_57a32d5.unitypackage)（[更新日志](/versions)）
<br/><a id='click'>    </a>
将插件导入Unity<br/>
执行以下步骤导入下载的插件:<br/>
在Unity中，选择Assets > Import Package > Custom Package…<br/>
选择你下载的Unity Plugin文件。<br/>
在Import Unity Package对话框中，单击Import。
