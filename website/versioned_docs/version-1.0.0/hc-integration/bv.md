## 一、加载Banner广告

调用SDK初始化后，SDK内部自动加载Banner广告。

## 二、展示Banner广告
```c
public void Button_ShowBanner()
{
    HCSDKManager.Instance.ShowBanner();
}
```

## 三、隐藏Banner广告
```c
public void Button_HideBanner()
{
    HCSDKManager.Instance.HideBanner();
}
```
